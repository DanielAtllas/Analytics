Servidores de Aplicação e Web - Analytics

Nos estudos sobre o mundo do BI e Big Data, haverá a grande possibilidade de deparação com os termos servidor de aplicação e servidor de aplicação web. Este terá como objetivo a introdução sobre esses termos e o que eles significam, além de onde cada um deve ser usado e se há alguma interação entre eles.

  • Servidor de aplicação

    • De forma geral, um servidor de aplicação é simplesmente um servidor no sentido da arquitetura cliente-servidor, onde é um processo que atende um ou mais aplicativos clientes que lhe enviam requisições, abrindo portas que atendam à conexões TCP e UDP. Além disso, para atender a requisições dinâmicas, o servidor de aplicação é composto por um servidor web e um container. Ou seja, o servidor de aplicação hospeda processos de um sistema que permitem clientes fazerem requisições e receberem respostas.

  • Servidor Web

    • Já um servidor Web, ou servidor de aplicação Web, possui um comportamento dinâmico, como páginas HTML personalizadas, arquivos, imagens e etc, contruidos com base em informação ou no resultado de uma consulta a um banco de dados. Ou seja, é uma aplicação destinado a atender necessidades web.
