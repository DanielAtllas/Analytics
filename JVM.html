JVM - Analytics

A JVM é máquina virtual Java (Java Virtual Machine) criada com o objetivo de rodar aplicações Java em "todos" os tipos de serviços e sistemas através de uma máquina virtual. Essa Virtual Machine é um software que simula uma máquina física e consegue executar programas, gerênciar processos, arquivos e memória totalmente virtuais (não dependendo de hardwares). Ou seja, para aplicações Java, não é necessário que suas execuções sejam dadas pelo sistema operacional em que a aplicação se encontra, sendo então acionadas diretamente com a JVM, possibilitando assim a portabilidade de seu código.

Esse processo cria uma independência do sistema operacional, fornecendo ao desenvolvedor a liberdade de desenvolver para múltiplas plataformas, ou seja, a preocupação de compilar o script e em qual sistema (Linux/Windows/Mac) o mesmo será gerenciado e executado, fica a par da JVM. Além disso, é responsável pela execução das pilhas, gerenciamento de memória, threads e etc.

Resumindo, o funcionamento é bem simples, onde um código é escrito em um arquivo .java, compilado e é gerado um arquivo .class conhecido como bytecode o ual é lido e interpretado pela JVM que posteriormente irá executa-lo. Dessa forma, um script feito no Linux e compilado no Linux pode ser executado em outros sistemas operacionais como o Windows e até mesmo em celulares, gerando assim a possibilidade de fazer uma aplicação Java rodar em qualquer sistema (multi-plataforma). Essa compilação do script para a geração do bytecode é realizado pelo JDK, o qual é kit de desenvolvimento Java responsável por esse processo de compilação.

Além disso, um programa não necessita se encontrar especificamente na linguagem Java para ser executado pela JVM. Outras linguagens que também gerem arquivos bytecode também podem rodar, como JRuby e Jython.

  • JRE - Ambiente de Execução Java

    • O JRE provê os requisitos mínimos para executar um programa Java, contendo uma JVM e os pacotes básicos do Java. Ou seja, é o que fornece as bibliotecas padrões do Java para o JDK compilar e a JVM executar.

  • JKD - Kit de Desenvolvimento Java

    • Este é o kit que provê ferramentas para desenvolvimento de programas Java, o qual contêm um compilador e o próprio JRE para a execução dos programas. Ou seja, é o JKD que compila os códigos transformando-os em bytecode.
