Cognos - Introdção - Analytics

O IBM Cognos Business Intelligence foi desenvolvido para construir análises e relatórios de application e BI. Diante disso, o tempo de vida de uma aplicação no Cognos pode levar de meses ou até mesmos anos, pois durante esse tempo, dados podem mudar e novos requisitos aparecerem. A medida que esses dados mudam, os autores devem analisar a mudança ou não de seus relatórios. Ou seja, o IBM Cognos é uma ferramenta para criação de relatórios e gráficos de dados do BI, tendo funcionalidades semelhantes ao Power BI e Grafana, o qual é desenvolvido pela IBM. Sendo assim, uma ferramenta análitica de relatórios e usaado para executar agregação de dados e criar relatórios detalhados compartilhados.

Estes relatórios podem ser compostos pr gráficos, múltiplas páginas, diferentes tabelas e prompts interativos, além desses relatórios poderem ser visualizados via navegadores web. Além disso, o Cognos oferece a opção de export de relatórios no formato XML ou PDF. É possível também programar os relatórios para rodarem em background em um específico horário e/ salvar o um relatório diários para não ser necessário roda-lo a todo momento.

Dessa forma, através do IBM Cognos Business Intelligence é possível analisar o desempenho dos negócios (através da criação de painéis/relatórios), descobrir o que impulsiona o desempenho do ambiente (de forma a descobrir insights nos dados coletados atráves de uma fácil linguagem, exploração visual e machine learning), compartilhamento dos insights (exibindo os painés de desempenho para terceitos) e seguranã/proteção dos dados (de forma a controlar o acesso com a governança baseada em função dos acessos).

Portanto, é uma ótima ferramenta para análise de dados BI, possibilintando converter informações dados em informações visuais

Primeiros passos no Cognos

Como dito anteriormente, o IBM Cognos é uma plataforma de criação de gráficos, painéis e dashboards para a visualização e análise de dados, com a possibilidade de inserir gráficos, tabelas e etc (exatamente como no Grafana e Power BI), sendo muito útil para ambientes com o uso de BI. Dessa forma, é possível fazer a importação de dados para o Cognos e neste tutorial, usare-mos de uma planilha excell obtida através de uma consulta SQL de uma tabela do SQL Server.

A tabela em questão, possui uma certa quantidade de colunas e linhas, assim como variações dos tipos de dados dessas colunas (desde caracteres até datas), o que irá ajudar na compreenção da criação do dashboard. Diante disso, para fazer a importação da planilha para o Cognos, vá na opção novo ou add source no canto inferior esquerdo e escolha a opção fazer upload de arquivos logo em seguida.

Feita a importação, será possível trabalhar na criação do dashboard indo na opção Painel no mesmo diretório que se encontra a opção de importação e então será aberta a janela de escolha do layout do dashboard. Várias opções se encontram disponpiveis, se diferenciando apenas pelo seus formatos e tamanhos (em caso de dúvidas, esolha a opção em full white ou 4 steps).

Realizado os passos anteriores, a janela de criação de dashboard será exibida, bastanto selecionar uma origem, a qual será o diretório onde foi salvada a planilha e ao fazer isso, serão exibidas as colunas presentes na planilha junto a seus respectvos dados. De forma geral, os dados já se ecnontram na plataforma, bastanto agora utilizar da criatividade/necessidade do dashboard a ser criado. No canto esquerdo da janela serão exibidas as opções de criação, como inserção de figuras, gráficos, tabelas e as próprias colunas presentes na source importada.

Para inserir um gráfico em uma determinada coluna, basta arrastar a coluna em questão para a tela de dashboard ou arrastar o tipo de gráfico desejado e depois arrastar a coluna para ele

Ao clicar no gráfico criado, várias opções de alteração serão exibidas, como inserção de título, ordem de visualização dos dados, cor e também a forma como esses dados serão exibidos. Assim como também é possível selecionar dois campos para mescla-los em um único gráfico (como por exemplo na criação de um gráfico por linhas que exibe a quantidade de registros realizados em uma determinada data). Para isso, basta usar da tecla ctrl e selecionar os campos desejados.

Ao arrastar um campo para o painel de dashboard, automaticamente a plataforma definirá um tipo de gráfico para aquele determinado dado, porém, também é possível alterar o formato em que esse gráfico será visível, basntando selecionar a opção selecionar visualização (além disso, a plataforma exibe os tipos de gráficos possíveis para a exibição de determinados dados. O nome de um usuário por exemplo, não é cabível em um gráfico numérico).

Outro aspect a ser levado em consideração, é a possibilidade de alteração das propriedades do dashboard a ser exibido, como cores do painél/gráficos/dados, formatos, exibições de alguns dados ou nome da coluna, basicamente, é possível alterar todo o corpo daquilo que será visualizado.
